---
name: telegram-notifier-setup
description: Safely set up, test, change, or remove local Telegram notifications for Codex without accepting a bot token in chat.
---

Never ask for a Telegram token in chat, a prompt, or a terminal command argument. Resolve the installed plugin root from this SKILL.md path (three parents up), never from the user's cwd. Run `node <plugin-root>/scripts/setup_server.js`; use `node <plugin-root>/scripts/codex_telegram_notifier.js status`, `test`, `remove`, or `doctor` as needed.
